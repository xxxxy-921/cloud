## ADDED Requirements

### Requirement: Login page acts as the visual source of system theme
The login page SHALL act as the visual source of the product's broader frontend theme. Its color language, surface layering, typography rhythm, and restrained brand emphasis SHALL establish the reference direction for the authenticated workspace rather than existing as an isolated one-off experience.

#### Scenario: Login page reviewed alongside workspace shell
- **WHEN** the login page and authenticated dashboard shell are reviewed together
- **THEN** they SHALL appear to belong to the same product family
- **AND** the shared identity SHALL come from tone, hierarchy, and token usage rather than copied hero effects
