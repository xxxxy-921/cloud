# 许可签发

许可签发用于创建、查询和维护许可全生命周期。你可以完成签发、导出 .lic、续期、暂停、恢复、吊销等操作，并查看许可详情中的注册码、状态与有效期。

## 一、功能概览

许可签发把“签发动作”和“生命周期管理”统一到一个流程里，帮助你持续管理已发放许可，而不是只做一次性创建。

当你需要发放新许可、处理存量许可变更、或导出交付文件时，应使用该模块。

## 二、功能清单

### 功能 1：签发新许可

功能介绍：
用于选择“商品”“授权主体”“套餐/自定义配置”，填写注册码与有效期，生成新许可记录。

使用场景：
- 新客户交付前签发首张许可
- 现有主体新增产品能力时补发许可

### 功能 2：筛选与查看许可记录

功能介绍：
用于按生命周期状态（待生效、生效中、已过期、已暂停、已吊销）筛选列表，并进入详情页查看关键信息。

使用场景：
- 按状态盘点在用许可
- 处理续期前先定位即将过期许可

### 功能 3：导出 .lic 交付文件

功能介绍：
用于从列表或详情页导出 .lic 文件，供客户端安装或离线验证。

使用场景：
- 交付实施阶段发放许可文件
- 重新下发历史许可文件

### 功能 4：管理生命周期操作

功能介绍：
用于执行“续期”“暂停”“恢复”“吊销”等操作，控制许可是否可继续使用。

使用场景：
- 合同续签后延长有效期
- 临时停用后恢复服务
- 永久终止时吊销许可

## 三、常见任务操作

### 任务 1：签发一条新许可

路径：许可管理 > 许可签发 > 签发许可

步骤：
1. 单击“签发许可”。
2. 依次选择“商品”“授权主体”“套餐或自定义”。
3. 选择或生成“注册码”，设置“生效日期/过期日期”。
4. 单击“签发”。

结果：
- 新许可写入列表，可进入详情查看。

影响：
- 会新增可交付的许可数据。

可撤销性：
- 不能直接删除；可通过“吊销”终止后续使用。

### 任务 2：导出 .lic 文件

路径：许可管理 > 许可签发 > 列表操作 或 许可详情 > 下载 .lic

步骤：
1. 在列表或详情中选择“导出 .lic / 下载 .lic”。
2. 保存文件到本地。
3. 将文件交付给目标环境使用。

结果：
- 生成可交付的 .lic 文件。

影响：
- 会影响外部交付流程与客户端激活。

可撤销性：
- 导出本身不可撤销，但可通过后续状态操作限制许可继续使用。

### 任务 3：吊销许可

路径：许可管理 > 许可签发 > 许可详情 > 吊销

步骤：
1. 打开目标许可详情页。
2. 单击“吊销”，在确认框中再次确认。
3. 刷新状态，确认变为“已吊销”。

结果：
- 许可进入“已吊销”状态。

影响：
- 直接影响许可可用性与后续导出策略。

可撤销性：
- 吊销不可恢复，请谨慎操作。

## 四、风险与注意事项

- 签发前请确认商品已发布、授权主体为活跃状态。
- 吊销是高风险操作，建议先完成业务确认再执行。
- 暂停适合临时管控，恢复后可继续使用。
- 导出文件涉及交付链路，建议做好版本与去向记录。
